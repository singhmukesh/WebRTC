What is it?

Communicator is a demo web application. 
Built on the Sencha Ext JS 6 web framework.
Integrated with TokBox's streaming video  technology.

Demo application with access to source code

Everything you need to make your own video chat application super fast!

Cloud Based Integration

Tokbox is a leading WebRTC platform for embedding live video, voice and messaging into websites.

Sencha Ext JS plays well with all sorts of back end services.

Integrating Tokbox with our enterprise class web application framework was a snap.

We've provided all the source code so you can see how we did it and guide you on using Ext JS in your own web application projects. 

Desktop Application

This demonstration uses cutting edge technology that not all web browsers yet support. 

Video camera access is currently supported in: Chrome, Firefox & Opera.

Safari & Internet Explorer will default to non-video camera usage.

Unsupported browsers will gracefully fall-back by removing the missing capabilities while still supporting real-time chat.
